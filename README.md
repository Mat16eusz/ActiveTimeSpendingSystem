# ActiveTimeSpendingSystem


## Usage
1. Clone the repository.
```
https://github.com/Mat16eusz/ActiveTimeSpendingSystem.git
```
2. Open the project in AndroidStudio.
3. Rename ``apikey.properties.example`` to ``apikey.properties`` and enter your BASE_URL, BASE_URL, SET_LOCAL_URL and MAPS_API_KEY in ``apikey.properties`` from your server domain and Google MAPS API console.
4. Start the application by pressing a button ``run "app"`` in AndroidStudio.
5. Application server available for cloning at the link.
```
https://github.com/Mat16eusz/ActiveTimeSpendingSystemApi.git
```
